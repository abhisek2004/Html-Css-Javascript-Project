# 🎓 AcadEase – Time Management Platform for B.Tech Students

AcadEase is a **student-centric academic time management and productivity website** designed specifically for **B.Tech / engineering students**.  
It brings together **planning, tracking, studying, and campus updates** into one clean, easy-to-use dashboard.

> **Built by students, for students** – because engineering is tough, your planner shouldn’t be.

---

## 🚀 Project Overview

**AcadEase** acts as a **one-stop solution** for managing:
- Classes & academic schedules
- Exams (previous & upcoming)
- Tasks & daily to-dos
- Study resources (Notes, PYQs, Assignments)
- Doubt solving (student–student & student–teacher)
- Campus announcements & events
- Productivity tools like **Pomodoro Timer**, **Study Timer**, and **GPA Calculator**

---

## 🧠 Key Features

### 🏠 Dashboard
- Today’s overview (classes, tasks, exams)
- Weekly progress tracking with visual bars
- Subject-wise progress insights

### 📅 Calendar
- Mini calendar view
- Academic, event, sports & hackathon highlights
- Year-long event planning (2026)

### 📝 Exams
- Separate sections for **Previous Exams** and **Upcoming Exams**
- Priority-based exam tracking
- Scroll-friendly responsive tables

### ✅ Tasks & To-Do
- Pending vs completed tasks
- Daily to-do list with:
  - Due date
  - Priority levels
- Weekly task completion analytics

### 📚 Resources
- Subject-wise:
  - Notes
  - Previous Year Questions (PYQs)
  - Assignments
  - Previous Question Papers
- Completion tracking for each resource type

### ❓ Doubts Section
- **Student–Teacher doubt raising system**
- **Student–Student discussion forum**
- Clear guidelines for effective doubt solving

### 📣 Announcements
- Filter by type (Tech, Sports, Academic, Hackathon, etc.)
- Sort by date or priority
- Priority-colored announcement banners

### ⚙️ Productivity Tools
- 📊 GPA Calculator
- ⏱️ Pomodoro Timer (25-5 method)
- ⏳ Custom Study Timer
- 🧠 Quiz Center (subject + difficulty based)

### 👤 Profile System
- Profile dropdown with:
  - Name
  - College email
  - Roll number
  - Branch, section, semester
- Logout option
- Theme toggle (Light / Dark)

## 🖥️ How to Run the Project

1. **Download or clone** this repository
2. Make sure all files are in the same folder
3. Open `index.html` in any modern web browser
4. No server or installation required ✅
